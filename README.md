# markbrucelatham.au

Personal website for Mark Latham — patient advocate, GCTB researcher, and patient navigator.

## Structure

- `index.html` — Homepage
- `my-story.html` — GCTB journey narrative
- `services.html` — Patient navigator services
- `research.html` — Research & advocacy work
- `contact.html` — Contact form and booking
- `css/style.css` — Stylesheet
- `js/main.js` — Minimal JavaScript
- `images/` — Photos and assets
- `CNAME` — Custom domain configuration

## Deployment

Hosted via GitHub Pages. Push to `main` and the site deploys automatically.

## Custom Domain

DNS is configured through Ventra IP to point `markbrucelatham.au` to GitHub Pages.

## Updating

Edit files directly and push. No build step required.
